About the Founder

Jace Glover is a young developer focused on making coding more accessible for students. AceBlocks was built from the idea that learning Roblox scripting should be visual, interactive, and beginner-friendly.

His goal is to build tools that help the next generation of creators start building games faster and with confidence.
