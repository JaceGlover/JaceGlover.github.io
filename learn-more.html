Learn More About AceBlocks

Making coding simple, visual, and accessible for every student.

What is AceBlocks?

AceBlocks is a visual coding platform designed to help students learn programming concepts through block-based coding, while also introducing real scripting (Luau for Roblox development).

Why AceBlocks?

  • Beginner-friendly block coding system
  • Bridges into real-world coding (Luau)
  • Designed for schools and classrooms
  • Supports creative game development

What You Can Learn

  • Programming fundamentals
  • Game development basics
  • Logic and problem-solving
  • Roblox scripting concepts

For Schools

AceBlocks is designed with education in mind, supporting structured lessons and aligning with digital technology learning outcomes. It can be used in classrooms, coding clubs, or independent learning.

Our Vision

We aim to make coding as approachable as possible, giving students the tools to build, experiment, and create their own games and projects.
